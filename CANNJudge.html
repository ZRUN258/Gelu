GELU算子

一、赛题背景

GELU（Gaussian Error Linear Unit，高斯误差线性单元）是深度学习中广泛使用的激活函数之一，由Dan Hendrycks和Kevin Gimpel于2016年提出。GELU将线性变换与标准正态分布的累积分布函数相结合，相比ReLU具有更平滑的非线性特性，在BERT、GPT、ViT等主流Transformer模型中表现优异，已成为大模型时代的核心激活函数。

本题要求基于PyTorch原生torch.nn.functional.gelu算子的核心业务逻辑，采用Ascend C编程语言进行算子原生开发，在昇腾NPU硬件上实现一款高性能、高精度、高适配性的GELU算子。

二、算子功能描述

实现的GELU算子需对输入张量逐元素计算GELU激活函数，返回与输入同形状的结果张量。

算子需支持float16（half）和float32（float）两种数据类型，兼顾计算精度与性能；需兼容非32整倍数的维度非对齐场景。

三、核心定义与约束

3.1 参考算子

PyTorch原生算子：torch.nn.functional.gelu / torch.nn.GELU（算子行为、计算结果需与该算子完全对齐）

3.2 数学公式

GELU激活函数的精确定义基于高斯累积分布函数：

GELU(x)=x⋅Φ(x)=x⋅12[1+erf(x2)]

其中 Φ(x) 为标准正态分布 N(0,1) 的累积分布函数，erf 为高斯误差函数：

erf(x)=2π∫0xe−t2dt

实现时可使用 Ascend C 提供的 erf 接口直接计算高斯误差函数，简化实现并保证数值精度。

3.3 输入输出与属性总览

类型参数名类型维度形状支持数据类型数据格式备注
INPUT(必选)selftensor(..., N4, N3, N2, N)float32, float16ND输入张量
OUTPUT(输出)outtensor(..., N4, N3, N2, N)与输入一致ND逐元素GELU激活后的结果张量，形状与输入一致

3.4 关键输入约束

  • 维度取值范围(均为正整数):
  • N ∈ [1, 10000]
  • N2 ∈ [1, 10000]
  • N3 ∈ [1, 1000]
  • N4 ∈ [1, 1000]
  • 输入为任意多维张量，最终维度可拆解为(..., N4, N3, N2, N)，前序...为任意合法批次维度，shape支持0-8维
  • 数据类型约束: 仅支持float32和float16两种浮点数据类型，不支持整型输入
  • 非对齐场景兼容: N、N2、N3、N4 均可能为非32的整倍数，算子需适配内存/数据非32字节对齐的场景
  • 数值取值约束: 输入张量的数值取值范围不超出各自数据类型的原生表达范围。float16需注意其有限的动态范围（约 ±65504）

3.5 输出严格要求

  • 形状约束: 输出形状与输入形状完全一致
  • 类型约束: 输出数据类型与输入数据类型一致
  • 数值含义: 输出张量中每个位置的值 = GELU(输入张量对应位置的值)
  • 精度要求: 要求与torch.nn.functional.gelu标杆精度误差满足浮点数精度要求
  • float32: 相对误差不超过 1e-4 或绝对误差不超过 1e-5
  • float16: 相对误差不超过 1e-3 或绝对误差不超过 1e-2（考虑FP16精度限制）

3.7 特殊值处理规则

  • NaN处理: 如果输入包含NaN，GELU(x)中x与NaN相乘/加结果仍为NaN，输出对应位置为NaN
  • Inf处理:
  • 如果输入为 +Inf: GELU(+∞)→+∞（erf(+∞)=1, 故结果为 +∞·1 = +∞）
  • 如果输入为 -Inf: GELU(−∞)→0−（erf(-∞)=-1, 故结果为 -∞·0 → 0，趋近于0的负值）
  • 极大正值处理: 当 |x| 较大（如 |x| > 6）时:
  • x → +∞: GELU(x) ≈ x，可直接返回x避免计算erf/exp
  • x → -∞: GELU(x) ≈ 0，可直接返回0
  • 此优化可在数值稳定性和计算效率之间取得平衡

四、规则要求

  1. 逐元素计算规则: 对输入张量的每个元素独立计算GELU激活函数，各元素间无数据依赖
  1. 精度对齐规则: 计算结果需与PyTorch torch.nn.functional.gelu 默认精确模式输出完全对齐
  1. 数据类型透传规则: 输出数据类型与输入数据类型保持一致，不做隐式类型转换
  1. 形状保持规则: 输出张量的形状与输入张量的形状完全一致，不做任何reshape或transpose操作
  1. 单调性保持规则: GELU函数为单调递增函数，输出结果需保持输入的相对大小关系（即若 x1>x2，则 GELU(x1)>GELU(x2)）

五、示例说明

示例1: 一维张量

  • 输入self: tensor([-2.0, -1.0, 0.0, 1.0, 2.0]), dtype=float32, shape=[5]
  • 输出out: tensor([-0.0455, -0.1587, 0.0000, 0.8413, 1.9545]), dtype=float32, shape=[5]
  • 结果解释:
  • GELU(-2.0) = -2.0 × 0.0228 ≈ -0.0455
  • GELU(0.0) = 0.0 × 0.5 = 0.0
  • GELU(2.0) = 2.0 × 0.9772 ≈ 1.9545

示例2: 二维张量，FP16

  • 输入self: tensor([[-1.0, 0.0], [1.0, 2.0]]), dtype=float16, shape=[2, 2]
  • 输出out: tensor([[-0.1587, 0.0000], [0.8413, 1.9545]]), dtype=float16, shape=[2, 2]
  • 结果解释: FP16模式下结果与FP32基本一致，精度损失在允许范围内

示例3: 特殊值处理（大输入）

  • 输入self: tensor([10.0, -10.0]), dtype=float32, shape=[2]
  • 输出out: tensor([10.0000, -0.0000]), dtype=float32, shape=[2]
  • 结果解释:
  • GELU(10.0) ≈ 10.0（Φ(10.0) ≈ 1，x ≈ GELU(x)）
  • GELU(-10.0) ≈ 0.0（Φ(-10.0) ≈ 0，输出趋近于0）

六、测试用例覆盖范围

  • 数据类型覆盖: float32, float16
  • 维度场景覆盖:
  • 0维（标量）场景
  • 1维、2维、3维、4维(N4,N3,N2,N)、含批次维度的多维场景(..., N4,N3,N2,N)
  • N~N4 为非32整倍数的非对齐场景
  • 数值场景覆盖:
  • 零值：验证 GELU(0) = 0
  • 正数范围：小正值（0 < x < 1）、中等正值（1 ≤ x < 6）、大正值（x ≥ 6）
  • 负数范围：小负值（-1 < x < 0）、中等负值（-6 < x ≤ -1）、大负值（x ≤ -6）
  • 标准正态分布随机数
  • 特殊值场景覆盖:
  • NaN值
  • +Inf、-Inf
  • 极大正值（验证数值稳定性）
  • 极小负值（验证趋近于0的正确性）
  • 精度对标覆盖: 与torch.nn.functional.gelu进行全场景精度对比：
  • float32: 绝对误差 < 1e-5，相对误差 < 1e-4
  • float16: 绝对误差 < 1e-2，相对误差 < 1e-3
